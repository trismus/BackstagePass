# Mitgliederverwaltung

> **Hinweis:** Dieser Bereich ist nur für **Vorstand** und **Administratoren** zugänglich.

---

## Übersicht

Die Mitgliederverwaltung ermöglicht:

- Mitglieder anzeigen und suchen
- Neue Mitglieder hinzufügen
- Mitgliederdaten bearbeiten
- Rollen und Status verwalten

---

## Mitgliederliste

### Aufrufen:
1. Gehe zu **"Mitglieder"** im Menü
2. Die Liste aller Mitglieder wird angezeigt

### Spalten:
| Spalte | Beschreibung |
|--------|--------------|
| **Name** | Vor- und Nachname |
| **E-Mail** | Kontakt-E-Mail |
| **Rolle** | Vereinsrolle |
| **Status** | Aktiv/Passiv |
| **Mitglied seit** | Eintrittsdatum |

### Filtern und Suchen:
- **Suche** - Nach Name oder E-Mail
- **Status** - Aktive/Passive Mitglieder
- **Rolle** - Nach Vereinsrolle filtern

---

## Neues Mitglied hinzufügen

### Schritt für Schritt:

1. Gehe zu **"Mitglieder"**
2. Klicke auf **"Neues Mitglied"** (+ Symbol)
3. Fülle die Pflichtfelder aus:
   - Vorname
   - Nachname
   - E-Mail-Adresse
4. Optional:
   - Telefonnummer
   - Adresse
   - Geburtsdatum
5. Wähle die **Rolle** (Standard: Mitglied Aktiv)
6. Klicke auf **"Speichern"**

### Nach dem Erstellen:
- Das Mitglied erhält eine Einladungs-E-Mail
- Es kann sein Passwort selbst setzen
- Das Profil kann vervollständigt werden

---

## Mitglied bearbeiten

### So bearbeitest du ein Mitglied:

1. Finde das Mitglied in der Liste
2. Klicke auf den Namen oder das Bearbeiten-Symbol
3. Ändere die gewünschten Felder
4. Klicke auf **"Speichern"**

### Bearbeitbare Felder:
- Kontaktdaten (Name, E-Mail, Telefon)
- Adresse
- Vereinsrolle
- Mitgliedsstatus

---

## Vereinsrollen

| Rolle | Beschreibung | Berechtigungen |
|-------|--------------|----------------|
| **Administrator** | System-Administrator | Voller Zugriff |
| **Vorstand** | Vereinsvorstand | Verwaltung, Events |
| **Mitglied Aktiv** | Aktives Mitglied | Anmeldungen, Proben |
| **Mitglied Passiv** | Passives Mitglied | Eingeschränkt |
| **Helfer** | Externer Helfer | Nur Helfereinsätze |
| **Partner** | Partner-Organisation | Partner-Bereich |
| **Freunde** | Vereinsfreunde | Minimal |

### Rolle ändern:
1. Öffne das Mitglied
2. Wähle die neue Rolle
3. Speichere die Änderung
4. Die Berechtigungen werden sofort aktualisiert

---

## Mitgliedsstatus

### Aktiv:
- Volles Stimmrecht
- Kann an allem teilnehmen
- Hat Soll-Stunden (falls definiert)

### Passiv:
- Eingeschränkte Rechte
- Keine Soll-Stunden
- Reduzierter Mitgliederbeitrag

### Status ändern:
1. Öffne das Mitglied
2. Ändere den Status
3. Speichere

---

## Mitglied deaktivieren

Bei Vereinsaustritt:

1. Öffne das Mitglied
2. Klicke auf **"Deaktivieren"**
3. Wähle den Grund:
   - Austritt auf eigenen Wunsch
   - Ausschluss
   - Verstorben
4. Bestätige die Aktion

> **Hinweis:** Deaktivierte Mitglieder können sich nicht mehr anmelden, bleiben aber im System für die Historie.

---

## Stundenkonto verwalten

Als Vorstand kannst du Stunden korrigieren:

1. Öffne das Mitglied
2. Gehe zu **"Stundenkonto"**
3. Klicke auf **"Buchung hinzufügen"**
4. Gib Datum, Stunden und Grund ein
5. Speichere

---

## Export

### Mitgliederliste exportieren:
1. Gehe zu **"Mitglieder"**
2. Klicke auf **"Exportieren"**
3. Wähle das Format (Excel, CSV, PDF)
4. Optional: Wähle Filter (nur aktive, bestimmte Rollen)
5. Lade die Datei herunter

---

## Häufige Fragen

### Ein Mitglied kann sich nicht anmelden
- Prüfe, ob das Konto aktiv ist
- Sende die Einladung erneut
- Setze das Passwort zurück

### Wie lösche ich ein Mitglied komplett?
- Mitglieder werden nur deaktiviert, nicht gelöscht
- Dies ist für die Historie und Buchhaltung notwendig

### Kann ich mehrere Mitglieder gleichzeitig bearbeiten?
- Nicht direkt, aber über den CSV-Import möglich
