# Events erstellen

> **Hinweis:** Dieser Bereich ist nur für **Vorstand** und **Administratoren** zugänglich.

---

## Übersicht

Als Vorstand kannst du verschiedene Veranstaltungen erstellen und verwalten.

---

## Event erstellen

### Schritt für Schritt:

1. Gehe zu **"Veranstaltungen"**
2. Klicke auf **"Neues Event"** (+ Symbol)
3. Wähle den **Event-Typ**:
   - Vereinsanlass
   - Helfereinsatz
   - Aufführung
4. Fülle die Details aus
5. Klicke auf **"Erstellen"**

---

## Event-Details

### Pflichtfelder:

| Feld | Beschreibung |
|------|--------------|
| **Titel** | Name der Veranstaltung |
| **Datum** | Wann findet es statt |
| **Beginn** | Startzeit |
| **Ende** | Endzeit |
| **Ort** | Veranstaltungsort |

### Optionale Felder:

| Feld | Beschreibung |
|------|--------------|
| **Beschreibung** | Details zum Event |
| **Max. Teilnehmer** | Begrenzung der Anmeldungen |
| **Anmeldefrist** | Bis wann kann man sich anmelden |
| **Sichtbarkeit** | Wer sieht das Event |

---

## Helfereinsätze erstellen

### Zusätzliche Felder:

| Feld | Beschreibung |
|------|--------------|
| **Partner** | Bei wem findet der Einsatz statt |
| **Vergütung** | Was erhält der Verein |
| **Einsatzleiter** | Wer ist verantwortlich |
| **Dress Code** | Kleidungsvorschriften |

### Schichten definieren:

1. Klicke auf **"Schicht hinzufügen"**
2. Definiere:
   - Name (z.B. "Aufbau")
   - Startzeit
   - Endzeit
   - Benötigte Helfer
3. Wiederhole für weitere Schichten

---

## Aufführungen erstellen

### Verknüpfung mit Stück:

1. Wähle das **Stück** aus der Liste
2. Oder erstelle ein neues Stück

### Schichten für Aufführungen:

Typische Schichten:
- **Garderobe** - Vor und während Aufführung
- **Bar/Buffet** - Vor, während, nach
- **Technik** - Licht, Ton
- **Einlass** - Ticketkontrolle
- **Auf-/Abbau** - Vor und nach

---

## Teilnehmer einladen

### Alle einladen:
1. Klicke auf **"Teilnehmer einladen"**
2. Wähle **"Alle Mitglieder"**
3. Die Einladung wird versendet

### Bestimmte Gruppen einladen:
1. Klicke auf **"Teilnehmer einladen"**
2. Filtere nach:
   - Rolle (nur Aktive, nur Helfer)
   - Status
   - Benutzerdefiniert
3. Versende die Einladungen

### Einzelne Personen einladen:
1. Suche die Person
2. Klicke auf **"Einladen"**

---

## Anmeldungen verwalten

### Anmeldungen ansehen:
1. Öffne das Event
2. Gehe zu **"Anmeldungen"**
3. Sieh alle Zu- und Absagen

### Status ändern:
1. Finde die Person in der Liste
2. Ändere den Status (Zugesagt/Abgesagt/Erschienen)
3. Speichere

### Nachrücker von Warteliste:
1. Öffne die Warteliste
2. Klicke auf **"Nachrücken lassen"**
3. Die Person wird benachrichtigt

---

## Event bearbeiten

### Änderungen vornehmen:
1. Öffne das Event
2. Klicke auf **"Bearbeiten"**
3. Ändere die gewünschten Felder
4. Speichere

### Teilnehmer benachrichtigen:
- Bei wichtigen Änderungen: Aktiviere **"Teilnehmer informieren"**
- Eine E-Mail wird automatisch versendet

---

## Event absagen/löschen

### Event absagen:
1. Öffne das Event
2. Klicke auf **"Event absagen"**
3. Gib einen Grund an
4. Alle Teilnehmer werden informiert

### Event löschen:
1. Nur möglich, wenn noch keine Anmeldungen
2. Klicke auf **"Löschen"**
3. Bestätige die Aktion

---

## Wiederkehrende Events

### Serie erstellen:
1. Erstelle das erste Event
2. Aktiviere **"Wiederholen"**
3. Wähle:
   - Täglich
   - Wöchentlich
   - Monatlich
4. Definiere das Ende der Serie
5. Alle Events werden erstellt

---

## Häufige Fragen

### Kann ich Events kopieren?
- Ja, öffne ein Event und klicke auf **"Duplizieren"**

### Wie ändere ich ein Event für alle Teilnehmer?
- Bearbeite das Event
- Aktiviere die Benachrichtigung
- Alle werden per E-Mail informiert

### Was passiert bei Überbuchung?
- Neue Anmeldungen landen auf der Warteliste
- Oder: Erhöhe das Teilnehmerlimit
