# Proben planen

> **Hinweis:** Dieser Bereich ist nur für **Vorstand**, **Regie** und **Administratoren** zugänglich.

---

## Übersicht

Als Regie oder Vorstand planst du Proben für Theaterstücke.

---

## Neue Probe erstellen

### Schritt für Schritt:

1. Gehe zu **"Proben"**
2. Klicke auf **"Neue Probe"** (+ Symbol)
3. Wähle das **Stück**
4. Fülle die Details aus
5. Klicke auf **"Erstellen"**

### Oder über das Stück:

1. Gehe zu **"Stücke"** → Wähle ein Stück
2. Klicke auf **"Proben"**
3. Klicke auf **"Neue Probe"**

---

## Proben-Details

### Pflichtfelder:

| Feld | Beschreibung |
|------|--------------|
| **Stück** | Welches Stück wird geprobt |
| **Datum** | Wann findet die Probe statt |
| **Beginn** | Startzeit |
| **Ende** | Endzeit |
| **Ort** | Probenraum |

### Optionale Felder:

| Feld | Beschreibung |
|------|--------------|
| **Probenart** | Leseprobe, Szenenprobe, etc. |
| **Szenen** | Welche Szenen werden geprobt |
| **Regie-Notizen** | Hinweise für Teilnehmer |

---

## Szenen auswählen

### Szenen zur Probe hinzufügen:

1. Klicke auf **"Szenen auswählen"**
2. Markiere die gewünschten Szenen
3. Speichere

### Auswirkungen:
- Beteiligte Rollen werden automatisch erkannt
- Besetzte Mitglieder werden eingeladen
- Die Probe erscheint in deren Kalender

---

## Teilnehmer automatisch einladen

Basierend auf den ausgewählten Szenen:

1. Das System erkennt alle beteiligten **Rollen**
2. Die **Besetzungen** werden ermittelt
3. Einladungen werden versendet

### Manuelle Anpassung:
- Zusätzliche Personen einladen
- Personen ausladen
- Alle einladen (Ensemble-Probe)

---

## Probenplan erstellen

### Mehrere Proben auf einmal:

1. Gehe zu **"Proben"** → **"Probenplan"**
2. Wähle den Zeitraum
3. Trage Proben im Kalender ein
4. Definiere für jede Probe die Szenen
5. Speichere den gesamten Plan

### Probenplan importieren:
- CSV-Import möglich
- Format: Datum, Zeit, Ort, Szenen

---

## Proben-Serien

### Wiederkehrende Proben:

1. Erstelle eine Probe
2. Aktiviere **"Wiederholen"**
3. Wähle:
   - Wöchentlich (z.B. jeden Dienstag)
   - Mehrmals pro Woche
4. Definiere das Ende
5. Alle Proben werden erstellt

### Serie bearbeiten:
- **Nur diese** - Ändert nur eine Probe
- **Alle zukünftigen** - Ändert diese und alle folgenden

---

## Proben verwalten

### Anwesenheit erfassen:

1. Öffne die Probe
2. Gehe zu **"Teilnehmer"**
3. Markiere:
   - ✅ Erschienen
   - ❌ Gefehlt
   - ⚪ Entschuldigt

### Regie-Notizen hinzufügen:

1. Öffne die Probe (nach der Durchführung)
2. Klicke auf **"Notizen hinzufügen"**
3. Schreibe allgemeine oder persönliche Notizen
4. Speichere

---

## Probe verschieben

### Termin ändern:

1. Öffne die Probe
2. Klicke auf **"Bearbeiten"**
3. Ändere Datum/Zeit/Ort
4. Aktiviere **"Teilnehmer informieren"**
5. Speichere

### Drag & Drop (Kalender):
- In der Kalenderansicht
- Ziehe die Probe auf einen neuen Tag
- Bestätige die Änderung

---

## Probe absagen

1. Öffne die Probe
2. Klicke auf **"Absagen"**
3. Gib einen Grund an (optional)
4. Alle Teilnehmer werden informiert

---

## Probenfortschritt

### Szenen-Status tracken:

Jede Szene hat einen Status:
- **Offen** - Noch nicht geprobt
- **In Arbeit** - Wird aktuell geprobt
- **Fertig** - Szene sitzt

### Status aktualisieren:
1. Gehe zum Stück → **"Szenen"**
2. Ändere den Status
3. Speichere

### Fortschrittsanzeige:
- Prozent der fertigen Szenen
- Visualisierung im Stück-Überblick

---

## Häufige Fragen

### Wie sehe ich, wer Zeit hat?
- Geplant: Verfügbarkeitsabfrage
- Aktuell: Prüfe die Absagen nach Erstellung

### Kann ich Proben kopieren?
- Ja, öffne eine Probe und klicke auf **"Duplizieren"**

### Wie plane ich eine Generalprobe?
- Wähle alle Szenen aus
- Setze Probenart auf "Generalprobe"
- Lade alle Beteiligten ein
