# Anmeldung & Profil

## Erstmalige Anmeldung

Wenn du zum ersten Mal BackstagePass nutzt, erhältst du eine Einladungs-E-Mail mit einem Link zur Registrierung.

### Schritte zur Registrierung:

1. **E-Mail öffnen** - Klicke auf den Link in der Einladungs-E-Mail
2. **Passwort setzen** - Wähle ein sicheres Passwort (mind. 8 Zeichen)
3. **Profil vervollständigen** - Fülle deine persönlichen Daten aus
4. **Fertig!** - Du wirst automatisch zum Dashboard weitergeleitet

---

## Anmeldung (Login)

### So meldest du dich an:

1. Öffne BackstagePass im Browser
2. Gib deine **E-Mail-Adresse** ein
3. Gib dein **Passwort** ein
4. Klicke auf **"Anmelden"**

> **Tipp:** Aktiviere "Angemeldet bleiben" um nicht jedes Mal das Passwort eingeben zu müssen.

---

## Passwort vergessen?

Falls du dein Passwort vergessen hast:

1. Klicke auf **"Passwort vergessen?"** auf der Login-Seite
2. Gib deine E-Mail-Adresse ein
3. Klicke auf **"Link senden"**
4. Öffne die E-Mail und klicke auf den Link
5. Setze ein neues Passwort

---

## Abmeldung (Logout)

Um dich abzumelden:

1. Klicke auf deinen Namen/E-Mail oben rechts
2. Wähle **"Abmelden"**

> **Wichtig:** Melde dich immer ab, wenn du einen geteilten Computer benutzt!

---

## Profil bearbeiten

Dein Profil enthält wichtige Informationen für den Verein:

### Persönliche Daten:
- Vorname und Nachname
- E-Mail-Adresse
- Telefonnummer
- Adresse

### So bearbeitest du dein Profil:

1. Klicke oben rechts auf deinen Namen
2. Wähle **"Profil"** oder **"Mein Bereich"**
3. Klicke auf **"Bearbeiten"**
4. Ändere die gewünschten Daten
5. Klicke auf **"Speichern"**

---

## Häufige Fragen

### Ich habe keine Einladungs-E-Mail erhalten
- Prüfe deinen Spam-Ordner
- Kontaktiere den Vorstand

### Mein Passwort funktioniert nicht
- Achte auf Gross-/Kleinschreibung
- Nutze "Passwort vergessen?" um ein neues zu setzen

### Ich möchte meine E-Mail-Adresse ändern
- Kontaktiere den Vorstand, da die E-Mail zur Anmeldung verwendet wird
